## 📱 RESPONSIVE BREAKPOINTS

| Breakpoint | Width | Description |
|------------|-------|-------------|
| `mobile` | < 640px | Single column, stacked layout |
| `tablet` | 640px - 1024px | 2 columns, sidebar collapse |
| `desktop` | > 1024px | Full layout, sidebar visible |

## 📝 NOTES FOR DEVELOPERS

1. **Mobile First**: Thiết kế mobile trước, sau đó mở rộng cho desktop
2. **Skeleton Loading**: Sử dụng Skeleton component khi loading data
3. **Error States**: Luôn handle empty state và error state
4. **Accessibility**: Đảm bảo keyboard navigation và screen reader support
5. **Image Optimization**: Sử dụng lazy loading cho images
